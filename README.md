Master Data Dashboard

Full Stack CRUD Application built using:

- React (Frontend)
- Python Flask (Backend)
- MySQL (Database)

Features

Units Module
- Add Unit
- Edit Unit
- Delete Unit
- View Units

Shifts Module
- Add Shift
- Edit Shift
- Delete Shift
- Dynamic row color rendering

Tech Stack
React | Flask | MySQL | REST APIs

How to Run

Backend
cd backend
python app.py

Frontend
cd frontend
npm install
npm start
